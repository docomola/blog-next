## 為什麼選擇 MDX？

MDX 是一種特殊的 Markdown 變體，支援嵌入式 JavaScript 和 JSX 語法。這使得您可以將 JavaScript 和 UI 元件[混合到 Markdown 內容中](https://docs.astro.build/en/guides/markdown-content/#mdx-features)，用於建立互動式圖表或提示框等功能。

如果您已有使用 MDX 創建的內容，此整合將使您輕鬆遷移到 Astro。
